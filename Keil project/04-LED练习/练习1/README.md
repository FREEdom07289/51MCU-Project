# 练习1：8灯亮1秒灭0.5秒

> 修正版：直接操作P2寄存器，纠正 `x = P2` 的错误

## 知识点

- `P2 = 0x00` → 8灯全亮
- `P2 = 0xFF` → 8灯全灭
- **易错**：`unsigned char x = P2;` 只拷贝了值，修改x不影响硬件

## 文件

| 文件 | 说明 |
|:---|:---|
| [main.c](main.c) | 修正后的正确代码 |

## 核心代码

```c
P2 = 0x00;    // 直接操作寄存器，8灯全亮
delay(1000);  // 持续1秒
P2 = 0xFF;    // 直接操作寄存器，8灯全灭
delay(500);   // 持续0.5秒
```
