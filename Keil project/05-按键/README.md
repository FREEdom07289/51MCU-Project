# 05-按键（初版）

> ⚠️ 当前为初版代码，包含典型错误 `unsigned char x = P2;`，用作学习对照

## 当前状态

- 代码实际上是LED控制（非按键检测）
- 包含**典型错误**：操作普通变量而非硬件寄存器

## 错误分析

```c
unsigned char x = P2;  // ❌ x是RAM变量，不是P2寄存器
x = 0x00;              // ❌ 修改x不影响硬件引脚
```

**正确写法**：直接操作 `P2 = 0x00;`
