# 06-独立按键控制LED亮灭

> 独立按键检测 + 软件消抖 + LED状态翻转

## 功能说明

- **K1（P3.1）**：每按一次，LED（D1）状态翻转一次（亮→灭→亮→灭…）
- 上电后LED默认熄灭，按下K1点亮，再按下熄灭，如此循环

## 硬件连接

| 外设 | 引脚 | 说明 |
|:---|:---|:---|
| LED（D1） | P2.0 | 低电平点亮（灌电流驱动） |
| 独立按键K1 | P3.1 | 按下为低电平，需软件消抖 |

## 运行效果

1. 上电后LED熄灭
2. **第1次按K1**：LED点亮
3. **第2次按K1**：LED熄灭
4. **第3次按K1**：LED点亮
5. 以此类推——每次按下翻转一次

## 代码解析

### 按键消抖标准流程

```c
if(KEY1 == 0)           // ① 检测按下
{
    delay_ms(20);       // ② 延时20ms等抖动过去
    if(KEY1 == 0)       // ③ 再次确认
    {
        while(KEY1 == 0); // ④ 等待松开（防连击）
        // 执行动作（如翻转LED）...
    }
}
```

### LED翻转核心语句

```c
LED1 = !LED1;   // 取反：1→0（亮）或 0→1（灭）
```

### 关键技术点

| 知识点 | 代码 | 说明 |
|:---|:---|:---|
| 消抖 | `delay_ms(20)` | 机械按键抖动约5-20ms，延时跳过抖动期 |
| 防连击 | `while(KEY1 == 0);` | 按着不松手只会触发一次 |
| 位操作 | `sbit LED1 = P2^0;` | 单独控制P2.0这一个引脚 |
| 状态翻转 | `LED1 = !LED1;` | 取反运算，亮灭交替 |
