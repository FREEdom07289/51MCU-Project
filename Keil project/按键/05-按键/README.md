# 05-按键

> 独立按键检测 + 软件消抖 + 按键控制LED流水灯

## 功能说明

- **K1（P3.0）**：切换LED流水方向（D1→D8 ↔ D8→D1）
- **K2（P3.1）**：启停流水灯（运行 ↔ 暂停）

## 硬件连接

| 外设 | 引脚 | 说明 |
|:---|:---|:---|
| LED（8个） | P2.0 ~ P2.7 | 低电平点亮（灌电流驱动） |
| 独立按键K1 | P3.0 | 按下为低电平，需软件消抖 |
| 独立按键K2 | P3.1 | 按下为低电平，需软件消抖 |

## 运行效果

1. 上电后LED从左往右流水（D1→D2→...→D8→循环）
2. **按K1**：流水方向立刻反转（D8→D7→...→D1）
3. **按K2**：流水灯暂停/继续（冻结在当前灯位置）
4. 按任意键都有消抖处理，不会误触发

## 代码解析

### 按键消抖标准流程

```c
if(KEY1 == 0)           // ① 检测按下
{
    delay_ms(20);       // ② 延时20ms等抖动过去
    if(KEY1 == 0)       // ③ 再次确认
    {
        while(KEY1 == 0); // ④ 等待松开（防连击）
        // 执行动作...
    }
}
```

### 关键技术点

| 知识点 | 代码 | 说明 |
|:---|:---|:---|
| 消抖 | `delay_ms(20)` | 机械按键抖动约5-20ms，延时跳过抖动期 |
| 防连击 | `while(KEY1 == 0);` | 按着不松手不会重复触发 |
| 方向控制 | `direction = !direction` | 布尔翻转，0↔1切换 |
| 启停控制 | `running = !running` | 运行/暂停状态切换 |
| 循环左移 | `_crol_(led, 1)` | D1→D2→...→D8 |
| 循环右移 | `_cror_(led, 1)` | D8→D7→...→D1 |

## 踩坑记录

### `unsigned char x = P2;` 错误（已修正）

初版代码犯了经典错误——把P2的值拷贝到普通变量x，后续修改x完全不影响硬件引脚。

| 错误写法 | 正确写法 | 原因 |
|:---|:---|:---|
| `unsigned char x = P2;` | 直接操作 `P2` | x是RAM变量，P2是硬件寄存器（地址0xA0） |
| `x = 0x00;` | `P2 = 0x00;` | 修改变量≠修改硬件 |
