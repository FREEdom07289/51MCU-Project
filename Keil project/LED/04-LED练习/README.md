## 练习1：8灯亮1s灭0.5s

**正确写法**：
```c
P2 = 0x00;      // 直接操作P2寄存器，8灯全亮
delay(1000);
P2 = 0xFF;      // 直接操作P2寄存器，8灯全灭
delay(500);
```

**常见错误**：`unsigned char x = P2; x = 0x00;` — x是RAM变量，不是硬件寄存器！

## 练习2：双灯流水灯

- 初始值 `0xFC`（1111 1100）→ D1、D2同时亮
- 循环7次（不是8次！第8次只剩单灯）
- 用 `_crol_` 循环左移，两个0一起走
