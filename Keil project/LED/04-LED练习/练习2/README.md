# 练习2：双灯流水灯

> 两个相邻的灯一起亮，像一双筷子从左扫到右

## 知识点

- 初始值 `0xFC = 1111 1100` → D1、D2同时亮
- `_crol_(x, 1)` 循环左移 → 两个0一起往高位移动
- **为什么循环7次？** 第8次会变成单灯（筷子断了）

## 逐帧动画

| 步数 | 二进制 | 亮灯 |
|:---:|:---|:---:|
| 初始 | `1111 1100` | D1 D2 |
| … | … | … |
| 第6步 | `0011 1111` | D7 D8 |
| → 重置 | `1111 1100` | 回到 D1 D2 |
