# 03-流水灯

> 3种方法实现流水灯效果，从底层位运算到库函数

## 实验目标

掌握循环移位原理，对比手写移位与 `_crol_`/`_cror_` 库函数的区别，最终实现双向来回流水灯。

## 知识点

| 知识点 | 说明 |
|:---|:---|
| `(x << 1) \| (x >> 7)` | 手写循环左移（底层原理） |
| `_crol_(x, 1)` | 库函数循环左移（需 `#include <intrins.h>`） |
| `_cror_(x, 1)` | 库函数循环右移 |
| `typedef unsigned char u8` | 类型重定义，简化代码 |
| `#define LED_PORT P2` | 宏定义端口别名 |

## 推荐学习顺序

```
手写移位 → 理解位运算原理
    ↓
库函数   → 一行替代，感受效率
    ↓
双向来回 → 组合运用，最佳实践
```

## 如何切换实现

Keil 工程中只能有一个 `main()` 函数。想测试其他实现时：

1. 把当前 `main.c` 改名为 `main-综合方法.c` 放入 `备选实现\`
2. 把 `备选实现\` 中想测试的文件复制到工程根目录，改名为 `main.c`
3. 在 Keil 中重新编译（Rebuild）

## 核心代码对比

```c
// 方法1：手写（理解原理）
x = (x << 1) | (x >> 7);

// 方法2：库函数（简洁）
x = _crol_(x, 1);

// 两者完全等价！
```
