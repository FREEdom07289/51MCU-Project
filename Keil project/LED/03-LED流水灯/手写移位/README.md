# 手写移位 — 流水灯

> 纯位运算实现双向流水灯，不依赖任何库函数

## 知识点

- 循环左移：`(x << 1) | (x >> 7)`
- 循环右移：`(x >> 1) | (x << 7)`
- 双向来回：先左扫7次，再右扫7次

## 文件

| 文件 | 说明 |
|:---|:---|
| [main-手写移位.c](main-手写移位.c) | 位运算版双向流水灯 |
| [STARTUP.A51](STARTUP.A51) | Keil启动文件 |

## 核心代码

```c
// 循环左移：最高位补到最低位
x = (x << 1) | (x >> 7);

// 循环右移：最低位补到最高位
x = (x >> 1) | (x << 7);
```
