# 51单片机按键输入

> 学完LED（输出），下一步学按键（输入）——从"让单片机说话"到"让单片机听你说话"的关键一步。

---

## 1. 按键基础原理

### 1.1 按键是什么？

- 按键就是一个**机械开关**
- 按下时电路导通 → 单片机引脚读到低电平（0）
- 松开时电路断开 → 单片机引脚读到高电平（1）

### 1.2 为什么需要消抖？

- 按键内部是**金属弹簧片**，按下瞬间不是稳稳闭合，而是在接触和断开间弹跳几次
- 弹跳持续时间：**5-20ms**
- 单片机很灵敏，会把每一次弹跳都当作一次"按下"
- 比喻 🏓：乒乓球落地弹几下才停——单片机听到了每次弹跳声，以为你按了好多次

### 1.3 消抖方案

| 方案 | 原理 | 普中A2使用 |
|:---|:---|:---|
| **软件消抖** | 检测到按下后，延时20ms再检测一次 | ✅ 最常用 |
| **硬件消抖** | 电容滤波或RS触发器 | 不常用 |

## 2. 初版代码的典型错误

> 📁 `Keil project/05-按键/main.c`（已修正，但作为学习案例保留分析）

### 2.1 错误代码

```c
#include <REGX52.H>

void delay(unsigned int t)
{
    int i, j;
    for(i = t; i > 0; i--)
    {
        for(j = 110; j > 0; j--);
    }
}

void main(void)
{
    unsigned char x = P2;   // ← 注意这一行！
    while(1)
    {
        x = 0x00;            // 修改的是x，不是P2
        delay(1000);
        x = 0xFF;            // 修改的是x，不是P2
        delay(500);
    }
}
```

### 2.2 🔍 代码分析

| 问题行 | 问题 | 后果 |
|:---|:---|:---|
| `unsigned char x = P2;` | x是RAM普通变量，仅拷贝了P2上电瞬间的数值 | 后续修改x无法改变硬件引脚电平 |
| `x = 0x00;` / `x = 0xFF;` | 赋值给了变量x，不是P2寄存器 | LED完全没反应 |
| `int i, j;` | 延时计数全为正数，用有符号int浪费 | 建议用 `unsigned int` |

### 2.3 ✅ 正确写法

```c
void main(void)
{
    while(1)
    {
        P2 = 0x00;       // ← 直接操作P2寄存器
        delay(1000);
        P2 = 0xFF;       // ← 直接操作P2寄存器
        delay(500);
    }
}
```

> 💡 **核心认知**：P2是硬件寄存器（地址0xA0），不是普通变量。修改x只是改RAM里的数值，和硬件完全无关。

## 3. 软件消抖标准模板

```c
#include <reg52.h>

sbit KEY1 = P3^0;   // 普中A2独立按键K1接P3.0
sbit LED1 = P2^0;

void delay_ms(unsigned int z)
{
    unsigned int i, j;
    for(i = z; i > 0; i--)
        for(j = 110; j > 0; j--);
}

void main()
{
    while(1)
    {
        if(KEY1 == 0)           // ① 检测到按键按下（低电平）
        {
            delay_ms(20);       // ② 延时20ms，等抖动过去
            if(KEY1 == 0)       // ③ 再次确认按键是否仍然按下
            {
                LED1 = ~LED1;   // ④ LED状态翻转
                while(KEY1 == 0); // ⑤ 等待按键松开（防止按一次触发多次）
            }
        }
    }
}
```

### 3.1 消抖流程说明

```
按键按下 → 等待20ms消抖 → 再次确认 → 执行动作 → 等待松开 → 回到检测
   │                         │
   └─ 抖动期（忽略）         └─ 稳定期（确认有效）
```

## 4. 实战：按键控制LED流水灯

> 📁 对应代码：`Keil project/05-按键/main.c`（✅ 已完成）

### 4.1 功能

| 按键 | 引脚 | 功能 |
|:---|:---|:---|
| K1 | P3.0 | 切换流水方向（D1→D8 ↔ D8→D1） |
| K2 | P3.1 | 启停流水灯（运行 ↔ 暂停） |

### 4.2 核心代码结构

```c
// 全局状态变量
u8 direction = 0;   // 0=左移，1=右移
u8 running   = 1;   // 1=运行，0=暂停

void main()
{
    u8 led = 0xFE;              // 初始：只有D1亮
    while(1)
    {
        u8 key = key_scan();    // 扫描按键（内部已消抖）

        if(key == 1)  direction = !direction;  // K1：切换方向
        if(key == 2)  running   = !running;    // K2：启停

        if(running)   // 运行中才执行流水
        {
            LED_PORT = led;
            delay_ms(200);
            led = (direction == 0) ? _crol_(led, 1) : _cror_(led, 1);
        }
    }
}
```

### 4.3 按键扫描函数（封装版）

```c
u8 key_scan(void)
{
    if(KEY1 == 0)
    {
        delay_ms(20);           // 消抖
        if(KEY1 == 0)
        {
            while(KEY1 == 0);   // 等松开（防连击）
            return 1;
        }
    }
    if(KEY2 == 0)
    {
        delay_ms(20);
        if(KEY2 == 0)
        {
            while(KEY2 == 0);
            return 2;
        }
    }
    return 0;   // 无按键
}
```
