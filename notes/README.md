## 🗺️ 建议学习路径

```
第1步：LED（输出）
  ├─ 点亮单个LED → 理解低电平驱动原理
  ├─ LED闪烁 → 理解延时函数
  └─ 流水灯 → 理解位运算 / _crol_ 移位
       │
第2步：按键（输入）
  ├─ 独立按键检测 → 理解机械抖动
  ├─ 软件消抖 → 20ms延时确认
  └─ 按键控制LED翻转 → 输入+输出联动
       │
第3步：数码管（显示）
  ├─ 静态显示 → 段码表
  └─ 动态扫描 → 位选+段选 + 视觉暂留
       │
第4步：中断与定时器
  ├─ 外部中断 → 按键触发中断
  ├─ 定时器中断 → 精确计时
  └─ PWM → 呼吸灯
       │
第5步：串口通信
  ├─ UART基础 → 波特率配置
  └─ 单片机与PC通信
       │
第6步：综合项目
  └─ 传感器采集 + 显示 + 控制
```
