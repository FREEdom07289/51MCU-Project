## 1. LED 模块

### 1.1 引脚连接

| LED | 正极（阳极） | 负极（阴极） | 控制引脚 |
|:---:|:---:|:---:|:---:|
| D1 | VCC（经R1限流） | P2.0 | P2.0 |
| D2 | VCC（经R2限流） | P2.1 | P2.1 |
| D3 | VCC（经R3限流） | P2.2 | P2.2 |
| D4 | VCC（经R4限流） | P2.3 | P2.3 |
| D5 | VCC（经R5限流） | P2.4 | P2.4 |
| D6 | VCC（经R6限流） | P2.5 | P2.5 |
| D7 | VCC（经R7限流） | P2.6 | P2.6 |
| D8 | VCC（经R8限流） | P2.7 | P2.7 |

### 1.2 电路原理

```
VCC（+5V）
   │
   ├─[R1 限流电阻]─[LED1 正极]─[LED1 负极]─── P2.0
   ├─[R2 限流电阻]─[LED2 正极]─[LED2 负极]─── P2.1
   ├─[R3 限流电阻]─[LED3 正极]─[LED3 负极]─── P2.2
   ├─[R4 限流电阻]─[LED4 正极]─[LED4 负极]─── P2.3
   ├─[R5 限流电阻]─[LED5 正极]─[LED5 负极]─── P2.4
   ├─[R6 限流电阻]─[LED6 正极]─[LED6 负极]─── P2.5
   ├─[R7 限流电阻]─[LED7 正极]─[LED7 负极]─── P2.6
   └─[R8 限流电阻]─[LED8 正极]─[LED8 负极]─── P2.7
```

### 1.3 点亮方式

- **低电平点亮**：P2.x 输出 `0`（0V），电流从 VCC → 电阻 → LED → P2.x（灌电流）
- **高电平熄灭**：P2.x 输出 `1`（5V），两端无电压差，LED不亮
- **原因**：51单片机灌电流能力强（~20mA），拉电流能力弱（<1mA），LED需要3-5mA

## 5. 引脚冲突注意事项

普中A2开发板上**多个模块共用同一组P口引脚**，使用时需注意：

- **P2口**：同时连接 LED（P2.0-P2.7）和数码管位选（P2.2-P2.5）
- **练习建议**：单独练习某个模块时，拔掉其他模块的跳线帽，避免互相干扰
