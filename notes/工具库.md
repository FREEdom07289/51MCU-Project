# 51单片机工具库与头文件

> 整理 Keil C51 开发中常用的头文件及其函数说明。

---

## 1. `reg52.h` — 标准8052寄存器定义

### 1.1 作用

把单片机内部硬件寄存器的**物理地址**映射为C语言中可直接使用的**变量名**。

例如：`P2` 实际上对应单片机内部地址 `0xA0`，`reg52.h` 用 `sfr P2 = 0xA0;` 将其定义为C语言变量，让你能直接写 `P2 = 0xFE;` 而不是操作裸地址。

### 1.2 包含的主要内容

| 类型 | 示例 | 说明 |
|:---|:---|:---|
| **端口寄存器** | `P0`, `P1`, `P2`, `P3` | 4组8位并行I/O口 |
| **可位寻址位** | `P2^0`, `P2^1` ... `P2^7` | 配合 `sbit` 使用，单独控制某个引脚 |
| **定时器/计数器** | `TCON`, `TMOD`, `TL0`, `TH0`, `TL1`, `TH1` | 定时器0和定时器1的配置与数据寄存器 |
| **串口** | `SCON`, `SBUF` | 串行通信控制与数据缓冲 |
| **中断** | `IE`, `IP` | 中断使能和优先级控制 |
| **电源控制** | `PCON` | 电源管理（含波特率倍增位SMOD） |

### 1.3 关键定义（部分）

```c
// 端口寄存器（sfr = Special Function Register）
sfr P0 = 0x80;
sfr P1 = 0x90;
sfr P2 = 0xA0;
sfr P3 = 0xB0;

// 可位寻址的位定义
sbit P2_0 = P2^0;  // 等价于 sbit LED1 = P2^0;
sbit P2_1 = P2^1;
// ... 以此类推
```

### 1.4 使用示例

```c
#include <reg52.h>   // 必须先引入，否则P2等标识符未定义

void main()
{
    P2 = 0xFE;       // 直接用，因为reg52.h已经定义了P2
    while(1);
}
```

## 2. `intrins.h` — 内联汇编函数库

### 2.1 作用

提供一组**直接映射到CPU汇编指令**的函数，用于位操作和空操作。这些函数编译后就是单条汇编指令，执行效率极高。

### 2.2 常用函数一览

| 函数原型 | 对应汇编指令 | 作用 | 示例 |
|:---|:---:|:---|:---|
| `unsigned char _crol_(unsigned char c, unsigned char b)` | `RL A` | 将 c **循环左移** b 位 | `_crol_(0xFE, 1)` → `0xFD` |
| `unsigned char _cror_(unsigned char c, unsigned char b)` | `RR A` | 将 c **循环右移** b 位 | `_cror_(0x7F, 1)` → `0xBF` |
| `unsigned int _irol_(unsigned int i, unsigned char b)` | — | 16位循环左移 | `_irol_(0xFF00, 4)` |
| `unsigned int _iror_(unsigned int i, unsigned char b)` | — | 16位循环右移 | — |
| `unsigned long _lrol_(unsigned long l, unsigned char b)` | — | 32位循环左移 | — |
| `unsigned long _lror_(unsigned long l, unsigned char b)` | — | 32位循环右移 | — |
| `void _nop_()` | `NOP` | 空操作，延时1个机器周期 | 用于精确时序控制 |
| `bit _testbit_(bit b)` | `JB`/`JNB` | 测试位并清零 | — |

### 2.3 重点函数详解

#### `_crol_` — 循环左移（最常用）

```c
#include <intrins.h>

unsigned char val = 0xFE;      // 1111 1110
val = _crol_(val, 1);          // 1111 1101 (D1灭, D2亮)
val = _crol_(val, 2);          // 1111 0111 (一次移2位)
```

等价手写代码：`val = (val << 1) | (val >> 7);`

#### `_cror_` — 循环右移

```c
unsigned char val = 0x7F;      // 0111 1111
val = _cror_(val, 1);          // 1011 1111
```

等价手写代码：`val = (val >> 1) | (val << 7);`

#### `_nop_` — 空操作

```c
#include <intrins.h>

// 精确延时（用于DS18B20等对时序要求严格的器件）
_nop_();  // 空转1个机器周期（12MHz下约1μs）
_nop_();
```

### 2.4 ⚠️ 注意事项

- `_crol_` / `_cror_` 是**循环**移位，不是逻辑移位——移出去的位会从另一端补回来
- 这些函数仅适用于**Keil C51 编译器**，换到其他平台（STM32/IAR/GCC）需要改写
- **推荐学习策略**：先手写位运算理解原理，再用 `_crol_` 偷懒

## 3. 延时函数模板

### 3.1 毫秒级延时（11.0592MHz晶振，经验值）

```c
/**
 * @brief 毫秒级延时函数
 * @param z  参数值，z=1000 约等于1秒
 * @note  基于11.0592MHz晶振的经验值，内层循环110次≈1ms
 */
void delay_ms(unsigned int z)
{
    unsigned int i, j;
    for(i = z; i > 0; i--)
        for(j = 110; j > 0; j--);
}
```

### 3.2 微秒级延时（粗略）

```c
/**
 * @brief 微秒级延时（非常粗略，11.0592MHz下约10μs）
 */
void delay_10us(unsigned int t)
{
    unsigned int i;
    for(i = t; i > 0; i--)
        _nop_();  // 需要 #include <intrins.h>
}
```
