# 51单片机学习项目

> 基于普中A2开发板，跟随江科大视频系统学习

## 开发环境
- **芯片**：STC89C52RC
- **晶振**：11.0592MHz
- **IDE**：Keil uVision + VSCode
- **下载工具**：STC-ISP
