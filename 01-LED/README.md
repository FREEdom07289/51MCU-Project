# 01-LED 模块

> LED 控制实验集 —— 单片机入门第一课

## 模块概述

LED 控制是单片机学习的起点。本模块包含从"点亮一个LED"到"流水灯"的渐进式实验，帮助你理解 I/O 端口控制、软件延时、位运算、以及嵌入式程序的基本结构。

## 实验列表

| 编号 | 实验名称 | 难度 | 知识点 |
|------|---------|------|--------|
| [1-点亮LED](1-点亮LED/) | 点亮一个LED | ⭐ | `sbit`位定义、P2口操作、低电平驱动 |
| [2-LED闪烁](2-LED闪烁/) | LED闪烁 | ⭐ | 软件延时函数、`while(1)`循环、整组P2赋值 |
| 3-流水灯 | LED流水灯（3种方法） | ⭐⭐ | 手写循环移位、`_crol_`/`_cror_`、`typedef`/`#define` |

> 📁 流水灯代码在 `Keil project/1-3LED流水灯/`，含3个版本：
> - `main-手写移位.c` — `(x<<1)|(x>>7)` 纯位运算

## 核心原理速查

### 为什么 P2=0x00 全亮？
- `0x00` = `0000 0000`，8个引脚全为低电平（0V）
- VCC(+5V) → 电阻 → LED → P2.x(0V)，电流流过 → 亮

### 为什么 P2=0xFF 全灭？
- `0xFF` = `1111 1111`，8个引脚全为高电平（5V）
- VCC(+5V) → 电阻 → LED → P2.x(5V)，无电压差 → 不亮

### 循环左移怎么实现？
```c
x = (x << 1) | (x >> 7);   // 手写
x = _crol_(x, 1);          // 库函数（需 #include <intrins.h>）
```
