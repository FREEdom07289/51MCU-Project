# 01-LED 模块

> LED 控制实验集 —— 单片机入门第一课

## 模块概述

LED 控制是单片机学习的起点。本模块包含从"点亮一个LED"到"LED闪烁"的渐进式实验，帮助你理解 I/O 端口控制、软件延时、以及嵌入式程序的基本结构。

## 实验列表

| 编号 | 实验名称 | 难度 | 知识点 |
|------|---------|------|--------|
| [1-点亮LED](1-点亮LED/) | 点亮一个LED | ⭐ | P2口操作、sbit位定义、共阳极驱动 |
| [2-LED闪烁](2-LED闪烁/) | LED闪烁 | ⭐ | 软件延时、while(1)循环、I/O翻转控制 |

## 硬件信息

| 项目 | 说明 |
|------|------|
| 开发板 | 普中A2 |
| 芯片 | STC89C52RC |
| 晶振 | 11.0592MHz |
| LED类型 | 板载共阳极LED（低电平点亮） |
| 控制引脚 | P2.0 ~ P2.7（8个板载LED） |

## 前置知识

- C语言基础：变量、循环、函数
- 二进制与十六进制基本概念
- 了解单片机引脚的基本概念

## 学习路线建议

```
1-点亮LED  ──▶  2-LED闪烁  ──▶  下一步：按键控制LED
  (基础)         (引入延时)        (输入+输出)
```
