# LED点灯实验

## 功能说明
控制普中A2开发板上的LED0（P2.0引脚）常亮

## 硬件连接
- LED0 → P2.0（板载LED，低电平点亮）

## 运行效果
上电后LED0常亮

## 学习要点
- P2口寄存器操作
- sbit定义引脚
- 低电平点亮原理（普中A2 LED为共阳极接法）

## 代码说明
```c
#include <reg52.h>
sbit LED = P2^0;

void main()
{
    while(1)
    {
        LED = 0;  // 输出低电平，点亮LED
    }
}
```

## 扩展实验
- 修改P2.0为其他引脚，观察不同LED
- 改为LED闪烁（加延时）
